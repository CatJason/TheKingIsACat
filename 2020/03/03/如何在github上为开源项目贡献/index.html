如何在github上为开源项目贡献

Posted on 2020-03-03 | In GitHub |
Words count in article: 255 | Reading time ≈ 1

以tensorflow为例子介绍如何在github上为一个开源项目做出贡献，进行多人协作编程

Fork

在你需要的项目主页，点击右上角的fork：

选择你要保存到自己的哪一个仓库：

稍等fork完成：

这里This branch is even with xiyouLinuxJavaWeb15:master，表示我们与主分支目前没有任何的差异。

这样我们创建了一个新的tensorflow项目的分支。

把Fork后的远程仓库拉到本地

利用SSH将仓库克隆至本地

就可以在本地进行编辑改动。

提交Issue

当我们在本地使用项目时发现了任何问题，提交到分支仓库下Issue中，或邮件致电他人进行沟通。

Issue板块在项目的主分支仓库才有

提交PR

等待项目的开发者review你的PR，进行讨论，修改直到符合之后，merge进主分支。

PR板块在我们自己的项目的分支仓库
