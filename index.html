查找与排序的关系

排序的过程

  1. 选择元素/位置
  2. 根据元素查找位置/根据位置查找元素
  3. 排序

所以查找是排序的前提，不同的查找方式导致了不同的排序方式

查找

根据元素查找位置

  • 顺序查找

  • 折半查找

  • 分块查找

排序

根据元素查找位置

  • 插入排序
    • 直接插入排序

      顺序查找

    • 折半插入排序

      折半查找

    • 希尔排序

      分块查找

根据位置查找元素

  • 交换排序
    • 冒泡排序

      选择无序表表尾的位置

    • 快速排序

      分治思想，一趟排序递归选择多个位置，递归排序

  • 选择排序
    • 简单选择排序

      选择线性表中无序表表尾位置

    • 堆排序

      选择树形结构的根或其子树的根结点

    堆排序之所以容易出错，是因为它是上述唯一的已排序的有序部分会在排序过程中再次变为无序的排序算法，这也是堆排序的数据结构是树这种递归结构却不能使用分治递归排序的原因。

两种特殊的排序算法

  • 归并排序

    子表之间是无序的（分治思想中，子表之间是有序的）。

  • 基排序

    已排序的有序部分会在排序过程中再次变为无序，与堆排序不同，不用再次进行排序。

Python中常用的序列结构有列表/元组/字符串/字典/集合以及range等对象也支持类似的操作
列表/元组/字符串支持双向索引,即具有负索引
列表是Python中内置有序可变序列,列表的所有元素放在一堆括号[]中,并使用,号隔开

列表的常用方法

方法 说明
lst.append(x) 将元素添加至列表lst尾部
lst.extend(L) 将L中所有元素添加至列表lst尾部
lst.insert(index,x) 在列表lst指定位置index处添加元素x,该位置后面的所有元素后移一个位置
lst.remove(x) 在列表lst中删除首次出现的指定元素,该元素之后的所有元素前移一个位置
lst.pop([index]) 删除并返回lst中下标为index(默认为-1)的元素
lst.clear() 删除列表lst中所有元素,但保留列表对象
lst.index() 返回列表lst中第一个值为x的元素的下标,若不存在值为x的元素则抛出异常
lst.count(x) 返回制定元素x在列表lst中的出现次数
lst.reverse() 对列表lst所有元素进行逆序
lst.sort(key=None,reverse=False) 对列表lst中的元素进行排序,key用来指定排序一句,reverse决定升序(Flase)还是降序(True)
lst.copy() 返回列表lst的浅复制(1)

  • python采用的是基于值的内存管理模式，如果为不同变量赋值为相同值，这个值在内存中只有一份，多个变量指向同一块内存地址

Python和C语言之间的区别

Python

  • Python是一门跨平台、开源、免费的解释型高级动态编程语言，支持伪编译编译将python源程序转换为字节码来优化程序和提高运行速度
  • 支持使用py2exe、pyinstaller或cx_Freeze工具将python程序转化为二进制可执行文件
  • Python支持命令式编程，函数式编程
  • 完全支持面向对象程序设计
  • 语法简洁清晰，拥有大量的几乎支持所有领域应用开发的成熟扩展库
  • 胶水语言，可以把多种不同语言编写的程序融合到一起实现无缝连接，更好地发挥不同语言和工具的优势，满足不同应用领域的要求

Python 3.x和2.x的区别

  • print()函数代替print语句
  • 3.x默认使用UTF-8编码
  • 除运算/有区别3.x：1/2=0.5 <– 2.x：1/2=0
  • 异常处理有区别
  • 八进制表示：3.x 002432 <– 2.x 02432或O02432
  • 不等于运算符：3.x!= <– 2.x!=或<>
  • 数据类型
    3.x 无long，只有int；增加bytes对应2.x的8位串
    2.x long和int
  • 以.py作扩展名的文件是python源代码文件，由python.exe解释，可在控制台下运行
  • 以.pyc作扩展名的文件是python编译文件。.pyc文件是不能直接用文本编辑器进行编辑，其优点是.pyc文件的执行速度要快
  • .pyw是设计用来运行python纯CUI(图形界面)程序的
  • .pyo文件python-O源文件即可将python源程序编译成.pyo文件，但有时仍得采用.pyc作后缀，才能正常运行
  • .pyd文件是非Python，由其他编程语言“编程-编译”生成的Python扩展模块。python要导入.pyd文件.实际上是在.pyd 文件中封装了一个module。在python中使用时，把他当成module来用就可以了。即：import路径名modulename即可。路径名为.pyd文件所在的路径。

特性

  • python属于强类型编程语言。python解释器会根据赋值或运算来自动推断变量类型
  • python采用的是基于值的内存管理模式，如果为不同变量赋值为相同值，这个值在内存中只有一份，多个变量指向同一块内存地址
  • python具有自动内存管理功能，不过该del还是del好
  • 字符串：使用单引号、双引号或三引号括起来的符号系列成为字符串
